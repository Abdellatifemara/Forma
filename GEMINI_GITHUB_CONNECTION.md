# Connect Gemini Web to GitHub Repository

## Method 1: Import Code in Gemini Chat

1. Go to https://gemini.google.com (logged in as abdellatif505050@gmail.com)
2. Click the **"+"** button or **"Add file"** icon
3. Select **"Import code"** or **"Connect to GitHub"**
4. Authorize Gemini to access your GitHub account
5. Select repository: **Abdellatifemara/Forma**
6. Gemini now has full context of your codebase
