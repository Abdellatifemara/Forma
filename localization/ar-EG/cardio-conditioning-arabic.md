## 🏃‍♂️ أنواع الكارديو

### ١. LISS - كارديو ثابت خفيف الشدة
**Low Intensity Steady State**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💚 LISS - الكارديو الخفيف
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 الوصف:
تمرين كارديو بشدة خفيفة لفترة طويلة
زي المشي السريع أو العجلة براحة

⏱️ المدة: ٣٠-٦٠ دقيقة
💓 النبض: ٥٠-٦٥% من أقصى نبض
😤 المجهود: تقدر تتكلم عادي

✅ المميزات:
• سهل على المبتدئين
• مش بيأثر على تمارين الحديد
• بيحرق دهون
• بيحسن صحة القلب
• تقدر تعمله كل يوم

❌ العيوب:
• بياخد وقت طويل
• الحرق أقل من HIIT
• ممكن يبقى ممل

🎯 مناسب لـ:
• المبتدئين
• أيام الراحة النشطة
• كبار السن
• اللي عندهم إصابات
• اللي بيعملوا تمارين حديد شاقة
```

### ٢. MISS - كارديو ثابت متوسط الشدة
**Moderate Intensity Steady State**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💛 MISS - الكارديو المتوسط
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 الوصف:
تمرين كارديو بشدة متوسطة
زي الجري الخفيف أو العجلة بسرعة معقولة

⏱️ المدة: ٢٠-٤٥ دقيقة
💓 النبض: ٦٥-٧٥% من أقصى نبض
😤 المجهود: تقدر تتكلم بصعوبة

✅ المميزات:
• توازن بين الوقت والحرق
• بيحسن اللياقة بشكل ملحوظ
• مش صعب زي HIIT

❌ العيوب:
• ممكن يأثر على الاستشفاء
• مش مناسب كل يوم

🎯 مناسب لـ:
• المستوى المتوسط
• اللي عايز يحسن لياقته
• ٢-٣ مرات في الأسبوع
```

### ٣. HIIT - تدريب متقطع عالي الشدة
**High Intensity Interval Training**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❤️ HIIT - الكارديو العالي الشدة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 الوصف:
فترات شغل عالية الشدة + فترات راحة
بتدي كل اللي عندك في فترة الشغل

⏱️ المدة: ١٥-٣٠ دقيقة (مع الإحماء والتهدئة)
💓 النبض: ٨٠-٩٥% من أقصى نبض
😤 المجهود: مش هتقدر تتكلم

✅ المميزات:
• وقت أقل، نتائج أكتر
• بيرفع الأيض لساعات بعد التمرين (EPOC)
• بيحافظ على العضلات
• بيحسن اللياقة بسرعة

❌ العيوب:
• صعب جداً
• محتاج استشفاء
• مش مناسب للمبتدئين
• ممكن يأثر على تمارين الحديد

🎯 مناسب لـ:
• المتقدمين
• اللي وقتهم ضيق
• ١-٣ مرات في الأسبوع بس
```

### ٤. Tabata - تاباتا
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔥 Tabata - تاباتا
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 الوصف:
نوع من HIIT بتوقيت محدد
٢٠ ثانية شغل + ١٠ ثواني راحة × ٨ جولات

⏱️ المدة: ٤ دقايق (جولة واحدة)
💓 النبض: ٩٠-١٠٠% من أقصى نبض
😤 المجهود: أقصى مجهود ممكن!

📊 البروتوكول:
┌─────────────────────────────────┐
│  ٢٠ ثانية شغل بأقصى مجهود     │
│  ١٠ ثواني راحة                │
│  كرر ٨ مرات = ٤ دقايق         │
│  ممكن تعمل ٢-٤ جولات          │
│  راحة ١-٢ دقيقة بين الجولات   │
└─────────────────────────────────┘

🎯 تمارين مناسبة:
• Burpees
• Mountain climbers
• Jump squats
• High knees
• Sprint
```

## 🔢 حسابات مهمة

### حساب أقصى نبض للقلب (MHR)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💓 حساب أقصى نبض (Maximum Heart Rate)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📐 المعادلة البسيطة:
أقصى نبض = ٢٢٠ - عمرك

📐 المعادلة الأدق (Tanaka):
أقصى نبض = ٢٠٨ - (٠.٧ × عمرك)

📊 أمثلة:
┌────────────────────────────────────┐
│ العمر │ المعادلة البسيطة │ Tanaka  │
├────────────────────────────────────┤
│  ٢٠   │      ٢٠٠        │   ١٩٤   │
│  ٢٥   │      ١٩٥        │   ١٩١   │
│  ٣٠   │      ١٩٠        │   ١٨٧   │
│  ٣٥   │      ١٨٥        │   ١٨٤   │
│  ٤٠   │      ١٨٠        │   ١٨٠   │
│  ٤٥   │      ١٧٥        │   ١٧٧   │
│  ٥٠   │      ١٧٠        │   ١٧٣   │
└────────────────────────────────────┘
```

### مناطق النبض (Heart Rate Zones)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 مناطق النبض - Heart Rate Zones
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔵 Zone 1 - راحة نشطة (٥٠-٦٠%)
   مشي خفيف، استشفاء
   بتتكلم عادي جداً

🟢 Zone 2 - حرق الدهون (٦٠-٧٠%)
   مشي سريع، عجلة خفيفة
   بتتكلم بسهولة

🟡 Zone 3 - هوائي (٧٠-٨٠%)
   جري خفيف، كارديو متوسط
   بتتكلم بصعوبة

🟠 Zone 4 - عتبة لاهوائية (٨٠-٩٠%)
   جري سريع، HIIT
   كلام متقطع بس

🔴 Zone 5 - أقصى مجهود (٩٠-١٠٠%)
   سبرنت، تاباتا
   مش هتقدر تتكلم خالص

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 مثال لشخص عمره ٣٠ سنة (أقصى نبض ١٩٠):

Zone 1: ٩٥-١١٤ نبضة/دقيقة
Zone 2: ١١٤-١٣٣ نبضة/دقيقة
Zone 3: ١٣٣-١٥٢ نبضة/دقيقة
Zone 4: ١٥٢-١٧١ نبضة/دقيقة
Zone 5: ١٧١-١٩٠ نبضة/دقيقة
```

### مقياس المجهود (RPE)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📏 مقياس المجهود - RPE Scale
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

من ١ لـ ١٠:

1️⃣ = قاعد على الكنبة
2️⃣ = ماشي في البيت
3️⃣ = مشي خفيف
4️⃣ = مشي سريع
5️⃣ = جري خفيف
6️⃣ = جري متوسط
7️⃣ = جري سريع - بتلهث
8️⃣ = جري سريع جداً - صعب
9️⃣ = سبرنت - تقريباً أقصى مجهود
🔟 = أقصى مجهود ممكن!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 استخدم RPE لو مش لابس ساعة:

• LISS: RPE ٣-٤
• MISS: RPE ٥-٦
• HIIT (شغل): RPE ٨-١٠
• HIIT (راحة): RPE ٣-٤
```

```

🤔 الكارديو قبل ولا بعد الحديد؟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 الإجابة: على حسب هدفك!

🎯 لو هدفك بناء عضل:
┌────────────────────────────────────┐
│ ❌ متعملش كارديو قبل الحديد       │
│ ✅ اعمل كارديو بعد الحديد         │
│ ✅ أو في يوم منفصل                │
│                                    │
│ السبب:                             │
│ الكارديو قبل الحديد بيستنفذ       │
│ الطاقة وبيأثر على الأداء          │
└────────────────────────────────────┘

🎯 لو هدفك حرق دهون:
┌────────────────────────────────────┐
│ ✅ الكارديو بعد الحديد أحسن       │
│    (الجليكوجين مستنفذ = حرق دهون) │
│                                    │
│ ✅ أو كارديو الصبح على الريق      │
│    (Fasted Cardio - LISS فقط)     │
│                                    │
│ ⚠️ لو هتعمل HIIT:                 │
│    يوم منفصل أحسن                  │
└────────────────────────────────────┘

🎯 لو هدفك لياقة عامة:
┌────────────────────────────────────┐
│ ✅ ٥-١٠ دقايق كارديو خفيف قبل     │
│    (كإحماء)                        │
│ ✅ ١٥-٢٠ دقيقة بعد الحديد          │
└────────────────────────────────────┘

📊 ملخص:
• بناء عضل: كارديو بعد أو يوم منفصل
• حرق دهون: كارديو بعد الحديد
• HIIT: يوم منفصل دايماً
• لياقة: خفيف قبل + متوسط بعد
```

┌────────────────────────────────────┐

│ ✅ ٣٠-٦٠ دقيقة                    │
│                                    │
│ ⚠️ فترة محدودة (٤-٨ أسابيع)      │

│ ⚠️ كل بروتين كفاية                │

### Fasted Cardio - كارديو الصيام

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🌅 Fasted Cardio - كارديو على الريق
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 إيه هو؟
تمرين كارديو الصبح قبل الفطار
بعد صيام ٨-١٢ ساعة (النوم)

🔬 النظرية:
مخازن الجليكوجين فاضية
الجسم بيحرق دهون مباشرة

📊 الحقيقة العلمية:
┌────────────────────────────────────┐
│ ✅ بيحرق نسبة دهون أعلى أثناء    │
│    التمرين                         │
│                                    │
│ ⚠️ لكن الحرق الكلي في اليوم      │
│    تقريباً نفسه                    │
│                                    │
│ 🎯 الأهم: إجمالي السعرات في اليوم │
└────────────────────────────────────┘

✅ مناسب لـ:
• اللي بيحب يتمرن الصبح
• LISS فقط (مش HIIT)
• اللي مش بيجوع الصبح
• كوباية قهوة قبل التمرين OK

❌ مش مناسب لـ:
• مرضى السكر
• اللي بيحس بدوخة
• HIIT أو تمارين شاقة
• اللي عنده مشاكل ضغط

💡 نصايح:
• ٢٠-٣٠ دقيقة بس
• LISS خفيف (مشي سريع أو عجلة)
• اشرب ماية كتير قبل
• كل بروتين بعد التمرين
• لو حسيت بدوخة، وقف فوراً
```

### نصائح عامة للكارديو

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 نصائح ذهبية للكارديو
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔥 الإحماء والتهدئة:
• دايماً ابدأ بـ ٣-٥ دقايق إحماء خفيف
• دايماً اختم بـ ٢-٣ دقايق تهدئة
• متوقفش فجأة بعد كارديو شاق

💧 الماية:
• اشرب قبل، أثناء، وبعد
• ٥٠٠ مل قبل التمرين
• رشفات صغيرة أثناء
• ٥٠٠+ مل بعد

👟 الأحذية:
• البس حذاء جري مناسب
• غيره كل ٦ شهور
• متجريش بحذاء قديم مهترئ

📱 التتبع:
• استخدم ساعة أو تطبيق
• راقب النبض
• سجل تقدمك

🎧 الموسيقى:
• الموسيقى بتحسن الأداء
• اختار إيقاع مناسب للسرعة
• ١٢٠-١٤٠ BPM للجري

😤 التنفس:
• تنفس من البطن
• شهيق من الأنف والفم
• زفير من الفم
• متحبسش نفسك

🏃 التدرج:
• زود ١٠% بس في الأسبوع
• الجسم محتاج وقت يتأقلم
• الإصابات من الزيادة السريعة
```
