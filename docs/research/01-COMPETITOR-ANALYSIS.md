# WHAT WE MUST MATCH

Based on competitor analysis, FORMA MUST have:

1. **Exercise Library:** At least 500 exercises with video demos
2. **Workout Tracking:** Simple, tap-based set logging
3. **Rest Timer:** Auto-start between sets
4. **Progress History:** See past workouts easily
5. **Offline Mode:** Core features work without internet
6. **Arabic Support:** Full Arabic UI option

---

# WHAT WE CAN DIFFERENTIATE ON

1. **Egyptian Food Database** - Our unique advantage
2. **Trainer Marketplace** - Connect with local trainers
3. **Local Payment Methods** - Vodafone Cash, Fawry
4. **Community** - Built for Egyptian fitness community
5. **Price** - Competitive EGP pricing
