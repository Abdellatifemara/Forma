**Design Elements:**

## Typography

### Primary Font: Cairo

**Why Cairo:**
- Designed for Arabic and Latin
- Excellent readability on screens
- Modern, clean aesthetic
- Free (Google Fonts)

### Font Weights

| Weight | Usage |
|--------|-------|
| Cairo Regular (400) | Body text, descriptions |
| Cairo Medium (500) | Subheadings, labels |
| Cairo SemiBold (600) | Headings, emphasis |
| Cairo Bold (700) | Headlines, CTAs |

### Font Sizes (Mobile)

| Name | Size | Line Height | Usage |
|------|------|-------------|-------|
| H1 | 32px | 40px | Screen titles |

## Splash Screen Animation

### Sequence (2 seconds total)

```
0.0s - Screen is Forma Teal (#00D4AA)
0.3s - Logo fades in (white, centered)
0.8s - Logo pulses slightly (scale 1.0 ‚Üí 1.05 ‚Üí 1.0)
1.2s - "FORMA" wordmark fades in below logo
1.5s - Tagline fades in
2.0s - Transition to app (fade to main screen)
```

### Loading States

- Use skeleton screens (not spinners)
- Skeleton color: `#1A2744` (dark mode) or `#E5E7EB` (light mode)
- Subtle shimmer animation left-to-right
