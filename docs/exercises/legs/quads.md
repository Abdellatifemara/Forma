# Quadriceps Exercises (100+ Movements)

## Anatomy
The quadriceps femoris is a group of four muscles on the front of the thigh:

- **Rectus Femoris**: Crosses both hip and knee (hip flexion + knee extension)
- **Vastus Lateralis**: Outer quad (lateral), largest quad muscle
- **Vastus Medialis**: Inner quad (medial), "teardrop" muscle
- **Vastus Intermedius**: Deep middle, under rectus femoris

### Function
- **Primary**: Knee extension
- **Secondary**: Hip flexion (rectus femoris only)
- **Stabilization**: Knee joint stability, patellar tracking

## Quad-Specific Training Tips

### Foot Position Effects
| Position | Effect |
|----------|--------|
| Narrow stance | More quad emphasis |
| Wide stance | More glute/adductor |
| Toes forward | Quad focused |
| Toes out | More adductors |
| Heels elevated | Quad emphasis (reduced ankle requirement) |
| Feet forward (Smith) | Extreme quad isolation |

### Vastus Medialis (VMO) Emphasis
- Full ROM squats (below parallel)
- Terminal knee extension
- Sissy squats
- Leg extension with pause at lockout
- Step-downs (eccentric emphasis)

### Rectus Femoris Emphasis
- Hip flexion component exercises
- Sissy squats (hip extended)
- Hanging leg raises (indirect)
