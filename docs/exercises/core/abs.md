## Anatomy Overview

### Rectus Abdominis ("Six-Pack")
| Attribute | Details |
|-----------|---------|
| **Origin** | Pubic crest and symphysis |
| **Insertion** | Xiphoid process, costal cartilages 5-7 |
| **Function** | Trunk flexion (ribs to pelvis), posterior pelvic tilt, compression of abdomen |
| **Innervation** | Intercostal nerves T7-T12 |

### Key Points
1. **It's ONE muscle** - "Upper" and "lower" abs are the same muscle
2. **Different emphasis** - Crunches emphasize upper, leg raises emphasize lower
3. **Anti-extension function** - Also stabilizes spine against extension (planks, rollouts)
4. **Visible at low body fat** - Typically <12% men, <20% women

### Upper vs Lower Abs (Same Muscle, Different Emphasis)
| Movement | Emphasis | Examples |
|----------|----------|----------|
| Ribs to pelvis | Upper rectus abdominis | Crunches, cable crunch |
| Pelvis to ribs | Lower rectus abdominis | Reverse crunch, leg raises |
| Anti-extension | Full rectus abdominis | Planks, rollouts |

## Crunch Variations (Upper Abs Emphasis)

### AB001 - Basic Crunch
| Attribute | Details |
|-----------|---------|
| **Equipment** | Floor |
| **Difficulty** | Beginner |
| **Primary** | Rectus abdominis (upper) |
| **Secondary** | - |
| **Contraindications** | Neck issues (if pulling on head) |

**Form Cues:**
- Lie on back, knees bent, feet flat
- Hands behind head (don't pull on neck!)
- Curl shoulders off floor, lifting chest toward ceiling
- Focus on RIBS moving toward PELVIS
- Lower with control, don't rest head on floor

**Common Mistakes:**
- Pulling on neck → Fingertips behind ears only
- Full sit-up → Only lift shoulders 30 degrees
- Using momentum → Slow, controlled movement

## Rollout Variations (Anti-Extension)

### AB049 - Ab Wheel Rollout (Kneeling)
| Attribute | Details |
|-----------|---------|
| **Equipment** | Ab wheel |
| **Difficulty** | Intermediate |
| **Primary** | Rectus abdominis |
| **Secondary** | Lats, shoulders |
| **Contraindications** | Lower back issues |

**Form Cues:**
- Kneel, grip wheel handles
- Roll out slowly, keeping abs tight
- Go as far as you can control
- Pull back using abs, not hip flexors
- Maintain flat back throughout

**Common Mistakes:**
- Arching lower back → Stop before this happens
- Using arms to pull back → Abs do the work
- Going too far → Build up gradually

## Form Guides for Key Exercises

### Crunch - Detailed Form
1. **Setup:** Lie on back, knees bent, feet flat
2. **Hands:** Behind head or across chest
3. **Initiate:** Contract abs, curl shoulders up
4. **Range:** Lift shoulder blades 30 degrees off floor
5. **Peak:** Squeeze abs hard
6. **Lower:** Control descent, don't rest head

### Ab Wheel Rollout - Detailed Form
1. **Setup:** Kneel, grip wheel, arms straight
2. **Brace:** Engage core, slight posterior pelvic tilt
3. **Roll Out:** Extend arms forward slowly
4. **Depth:** Go only as far as you can control
5. **Return:** Pull back using abs, not arms
6. **Key:** Never let lower back arch!

### Hollow Body Hold - Detailed Form
1. **Start:** Lie flat, arms overhead
2. **Press:** Lower back must stay pressed to floor
3. **Lift:** Raise arms, head, shoulders, legs
4. **Shape:** Body forms banana/boat shape
5. **Hold:** Maintain position for time
6. **Scale:** Bend knees or tuck arms to make easier

## Common Mistakes & Fixes

| Mistake | Why It's Wrong | Fix |
|---------|---------------|-----|
| Pulling on neck during crunches | Neck strain | Fingertips behind ears only |
| Doing full sit-ups for abs | More hip flexor | Focus on crunches for abs |
| Letting back arch on leg raises | Lower back strain | Press back into floor |
| Going too fast | Uses momentum | Slow, controlled reps |
| Holding breath | Increases pressure | Breathe steadily |
| Only doing crunches | Incomplete development | Add planks, leg raises, rollouts |
| Ignoring anti-extension | Missing key function | Add planks, rollouts |
