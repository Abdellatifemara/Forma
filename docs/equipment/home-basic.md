# Basic Home Gym Equipment

## Essential Items

| Equipment | Cost (EGP) | Priority | Exercises Unlocked |
|-----------|------------|----------|-------------------|
| Adjustable Dumbbells (2-20kg) | 2000-4000 | 1 | 50+ exercises |
| Pull-up Bar (doorway) | 300-600 | 2 | Pull-ups, chin-ups, hanging |
| Resistance Bands (set) | 200-500 | 3 | All muscle groups |
| Exercise Mat | 150-400 | 4 | Floor exercises, stretching |
| Jump Rope | 50-150 | 5 | Cardio, warm-up |

## Dumbbells

### Weight Recommendations by Level
| Level | Men | Women |
|-------|-----|-------|
| Beginner | 5-15 kg pair | 2-8 kg pair |
| Intermediate | 10-25 kg pair | 5-15 kg pair |
| Advanced | Adjustable 5-35 kg | Adjustable 2-20 kg |

### Exercises Unlocked
- All pressing movements
- All rowing movements
- Curls and extensions
- Lunges, goblet squats
- Romanian deadlifts
- Shoulder work

## Pull-up Bar

### Types
| Type | Price | Pros | Cons |
|------|-------|------|------|
| Doorway (no screws) | 300-500 | Easy install | Width limited |
| Doorway (screws) | 400-700 | More stable | Permanent holes |
| Wall-mounted | 600-1200 | Most stable | Requires drilling |
| Free-standing | 1500-3000 | No installation | Takes floor space |

### Exercises
- Pull-ups (wide, close, neutral)
- Chin-ups
- Hanging leg raises
- Knee raises
- Dead hangs (grip strength)
- Scapular pulls

## Resistance Bands

### Band Types
| Color (typical) | Resistance | Use |
|-----------------|------------|-----|
| Yellow | 5-15 lbs | Warm-up, rehab |
| Red | 15-35 lbs | Light exercises |
| Green | 25-65 lbs | Medium resistance |
| Blue | 40-80 lbs | Heavy resistance |
| Black | 50-120 lbs | Pull-up assist, heavy |

### Key Uses
- Assisted pull-ups
- Band pull-aparts
- Face pulls
- Banded squats
- Tricep pushdowns
- Bicep curls

## Budget Setup (~3000 EGP)
1. Adjustable dumbbells (2000)
2. Doorway pull-up bar (400)
3. Resistance band set (300)
4. Exercise mat (200)
5. Jump rope (100)
