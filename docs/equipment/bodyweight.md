# Bodyweight Equipment (No Equipment)

## Overview
Training with zero equipment using only body weight as resistance.

## Available "Equipment"

| Item | Use | Notes |
|------|-----|-------|
| Floor | Push-ups, planks, core | Any flat surface |
| Wall | Wall sits, wall push-ups | For support/angles |
| Chair/Couch | Dips, step-ups, elevated push-ups | Sturdy furniture |
| Doorframe | Pull-up bar mounting, stretches | Check stability |
| Stairs | Step-ups, calf raises, cardio | Alternative to box |
| Towel | Rows, hamstring curls, stretches | Makeshift equipment |
| Backpack | Add weight to exercises | Fill with books/bottles |

## Progression Methods
1. Add reps (up to 25-30)
2. Slow tempo (4 sec eccentric)
3. Pause reps (2-3 sec at bottom)
4. Harder variation
5. Single-limb versions
6. Add backpack weight
7. Decrease rest time
