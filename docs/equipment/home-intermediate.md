# Intermediate Home Gym Equipment

## Recommended Additions

| Equipment | Cost (EGP) | Priority | Exercises Unlocked |
|-----------|------------|----------|-------------------|
| Adjustable Bench | 1500-3000 | 1 | Incline/decline pressing |
| Barbell + Weights | 3000-6000 | 2 | All compound lifts |
| Kettlebell (16-24kg) | 800-1500 | 3 | Swings, Turkish get-ups |
| Dip Station | 800-1500 | 4 | Dips, leg raises |
| Ab Wheel | 100-300 | 5 | Core rollouts |
| Foam Roller | 150-400 | 6 | Recovery, mobility |

## Adjustable Bench

### Features to Look For
- Flat to incline (0-90째)
- Decline option (bonus)
- Weight capacity 150kg+
- Stable base
- Comfortable padding

### Exercises Unlocked
| Position | Exercises |
|----------|-----------|
| Flat | Bench press, flyes, pullovers |
| Incline (30째) | Incline press, incline curls |
| Incline (45째) | Shoulder-emphasis press |
| Decline | Decline press, decline flyes |
| Seated (90째) | Shoulder press, face-level curls |

## Barbell Setup

### Minimum Package
| Item | Specification | Cost |
|------|---------------|------|
| Olympic Barbell | 20kg, 7ft | 1500-2500 |
| Weight Plates | 50-100kg total | 1500-3500 |
| Collars | Spring or clamp | 100-200 |

### Weight Plate Recommendation
| Plates Needed | Total Weight |
|---------------|--------------|
| 2x 20kg | 40kg |
| 2x 10kg | 20kg |
| 2x 5kg | 10kg |
| 4x 2.5kg | 10kg |
| 4x 1.25kg | 5kg |
| **Total** | **85kg + 20kg bar = 105kg** |

### Exercises Unlocked
- Barbell squat (need rack for heavy)
- Deadlift
- Barbell row
- Floor press
- Overhead press
- Barbell curl
- Romanian deadlift

## Kettlebell

### Recommended Starting Weights
| Level | Men | Women |
|-------|-----|-------|
| Beginner | 12-16 kg | 8-12 kg |
| Intermediate | 20-24 kg | 12-16 kg |
| Advanced | 28-32 kg | 20-24 kg |

### Key Exercises
- Kettlebell swing (posterior chain)
- Goblet squat
- Turkish get-up (full body)
- KB clean & press
- KB snatch
- KB row

## Dip Station

### Types
| Type | Price | Best For |
|------|-------|----------|
| Parallel bars | 800-1200 | Chest/tricep dips |
| Power tower | 2000-4000 | Dips + pull-ups + knee raises |
| Wall-mounted | 600-1000 | Space saving |

### Exercises
- Chest dips (forward lean)
- Tricep dips (upright)
- Hanging knee raises
- L-sit holds
- Inverted rows (low bars)

- Adjustable bench (2000)
- Barbell + 80kg plates (3500)
