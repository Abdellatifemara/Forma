# Calisthenics Program

## Overview
- **Frequency**: 4-6x per week
- **Duration**: 45-60 minutes
- **Equipment**: Pull-up bar, parallel bars (optional)
- **Goal**: Build strength and skills with bodyweight

## Skill Levels

### Beginner Skills
- Push-ups (10+)
- Inverted rows (10+)
- Squats (20+)
- Plank (60s)

### Intermediate Skills
- Pull-ups (10+)
- Dips (10+)
- Pike push-ups (10+)
- Pistol squat (assisted)
- L-sit (15s+)

### Advanced Skills
- Muscle-ups
- Handstand push-ups
- One-arm push-ups
- Pistol squats
- Front lever
- Back lever
- Planche (progressions)

## Skill Work Template (15 min)
| Time | Activity |
|------|----------|
| 0-5 min | Wrist warm-up, shoulder prep |
| 5-12 min | Main skill practice |
| 12-15 min | Active rest, flexibility |

**Focus on 1-2 skills per session**

## Rep/Set Guidelines

### For Strength
- 3-5 sets
- 3-8 reps
- 3-5 min rest
- Harder progressions

### For Muscle (Hypertrophy)
- 3-4 sets
- 8-15 reps
- 60-90 sec rest
- Moderate progressions

### For Endurance
- 2-3 sets
- 15-30 reps
- 30-60 sec rest
- Easier progressions

## Progression Rules
1. Hit top of rep range for 2 sessions
2. Move to harder variation
3. Reset reps at bottom of range
4. Build back up
5. Repeat
