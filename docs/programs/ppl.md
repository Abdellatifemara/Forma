# PPL - Push Pull Legs (6-Day Split)

## Overview
- **Frequency**: 6x per week
- **Duration**: 60-75 minutes
- **Level**: Intermediate (6+ months)
- **Goal**: Maximum muscle growth

## Weekly Schedule
| Day | Workout |
|-----|---------|
| Monday | Push A (Strength) |
| Tuesday | Pull A (Strength) |
| Wednesday | Legs A (Strength) |
| Thursday | Push B (Hypertrophy) |
| Friday | Pull B (Hypertrophy) |
| Saturday | Legs B (Hypertrophy) |
| Sunday | Rest |

| Exercise | Sets | Reps | Rest |
|----------|------|------|------|

## Progression
Two-for-Two Rule: When you hit top of rep range for 2 sessions, add weight.
- Upper body: +5 lbs
- Lower body: +10 lbs
