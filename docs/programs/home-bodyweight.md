# Home Bodyweight Program

## Overview
- **Frequency**: 5-6x per week
- **Duration**: 30-45 minutes
- **Equipment**: None (pull-up bar optional)
- **Goal**: Build muscle at home

## Weekly Schedule
| Day | Workout |
|-----|---------|
| Monday | Push |
| Tuesday | Pull |
| Wednesday | Rest or Cardio |
| Thursday | Legs |
| Friday | Upper Body |
| Saturday | Core & Cardio |
| Sunday | Rest |

## Push Day
| Exercise | Sets | Reps | Progression |
|----------|------|------|-------------|
| Push-Ups | 4 | 15-25 | → Diamond → Decline |
| Pike Push-Ups | 3 | 10-15 | → Elevated Pike |
| Diamond Push-Ups | 3 | 10-15 | → Close Decline |
| Chair Dips | 3 | 8-15 | → Weighted |
| Plank Shoulder Taps | 3 | 10/side | → Slower |

## Pull Day
| Exercise | Sets | Reps | Progression |
|----------|------|------|-------------|
| Pull-Ups/Chin-Ups | 4 | 5-12 | → Wide → L-Sit |
| Inverted Rows | 4 | 10-15 | → Feet elevated |
| Doorway Curls | 3 | 10-15 | → Slow eccentric |
| Back Extensions | 3 | 15-20 | → Superman |
| Australian Pull-Ups | 3 | 10-15 | → Single arm |

## Legs Day
| Exercise | Sets | Reps | Progression |
|----------|------|------|-------------|
| Squats | 4 | 15-25 | → Jump → Pistol |
| Bulgarian Split Squat | 3 | 10-15/leg | → Weighted |
| Single-Leg RDL | 3 | 10-12/leg | → Weighted |
| Glute Bridge | 3 | 15-20 | → Single-leg |
| Calf Raises | 4 | 20-30 | → Single-leg |
| Wall Sit | 3 | 30-60s | → Single-leg |

## Upper Body Full
| Exercise | Sets | Reps |
|----------|------|------|
| Push-Up Variation | 3 | Max |
| Pull-Up Variation | 3 | Max |
| Pike Push-Ups | 3 | 10-15 |
| Inverted Rows | 3 | 10-15 |
| Dips | 3 | 8-15 |
| Plank | 3 | 45-60s |

## Core & Cardio
| Exercise | Sets | Reps/Time |
|----------|------|-----------|
| Plank | 3 | 45-60s |
| Mountain Climbers | 3 | 30s |
| Bicycle Crunches | 3 | 20/side |
| Leg Raises | 3 | 15-20 |
| Burpees | 3 | 10-15 |
| Jump Squats | 3 | 15-20 |

## Progression Without Weights
1. Add reps (up to 25-30)
2. Add sets
3. Harder variation
4. Slower tempo (4 sec down)
5. Less rest time
6. Add backpack weight
7. Single-limb versions
