# Posture Correction Guide

## Common Postural Issues

### Forward Head Posture
**Signs:**
- Head protrudes forward
- Chin juts out
- Neck pain/tension

**Causes:**
- Phone use (text neck)
- Computer work
- Weak deep neck flexors

**Corrections:**
| Exercise | Sets | Reps/Time |
|----------|------|-----------|
| Chin tucks | 3 | 15 reps |
| Wall angels | 3 | 10 reps |
| Neck stretches | 2 | 30s each |
| Upper trap stretch | 2 | 30s each |

### Rounded Shoulders
**Signs:**
- Shoulders roll forward
- Upper back rounded
- Shoulder pain

**Causes:**
- Tight pecs
- Weak upper back
- Desk work

**Corrections:**
| Exercise | Sets | Reps/Time |
|----------|------|-----------|
| Face pulls | 3 | 15-20 |
| Band pull-aparts | 3 | 15-20 |
| Doorway stretch | 3 | 30s each |
| Rows | 3 | 12-15 |
| Rear delt flyes | 3 | 15 |

### Anterior Pelvic Tilt
**Signs:**
- Lower back arches excessively
- Belly sticks out
- Tight hip flexors

**Causes:**
- Sitting too much
- Weak glutes/abs
- Tight hip flexors

**Corrections:**
| Exercise | Sets | Reps/Time |
|----------|------|-----------|
| Hip flexor stretch | 3 | 45s each |
| Glute bridges | 3 | 15 |
| Dead bugs | 3 | 10 each |
| Plank | 3 | 30-45s |
| RKC plank | 3 | 15-20s |

### Posterior Pelvic Tilt
**Signs:**
- Flat lower back
- Tucked tailbone
- Tight hamstrings

**Causes:**
- Weak lower back
- Tight hamstrings
- Over-tucking habit

**Corrections:**
| Exercise | Sets | Reps/Time |
|----------|------|-----------|
| Hamstring stretch | 3 | 45s each |
| Hip hinge practice | 3 | 10 reps |
| Romanian deadlifts | 3 | 10-12 |
| Cat-cow | 2 | 10 reps |

## Posture Exercises

### Chin Tucks
**How to:**
1. Sit or stand tall
2. Pull chin straight back (make double chin)
3. Hold 5 seconds
4. Release and repeat
5. Keep eyes level

### Wall Angels
**How to:**
1. Back against wall
2. Feet 6 inches from wall
3. Arms in "goalpost" position against wall
4. Slide arms up and down
5. Keep contact with wall throughout

### Face Pulls
**How to:**
1. Cable at face height
2. Rope attachment
3. Pull toward face, elbows high
4. Separate rope at face
5. Squeeze shoulder blades
6. Slow controlled return

### Dead Bug
**How to:**
1. Lie on back, arms up
2. Knees bent 90°
3. Press lower back into floor
4. Extend opposite arm and leg
5. Return, switch sides
6. Keep back flat throughout

### RKC Plank
**How to:**
1. Forearm plank position
2. Squeeze glutes hard
3. Tighten quads
4. Pull elbows toward toes (don't move)
5. Create full body tension
6. Hold with maximum effort

## Posture Throughout the Day

### Sitting
- Feet flat on floor
- Knees at 90°
- Back supported
- Shoulders back
- Screen at eye level
- Elbows at 90°

### Standing
- Weight even on both feet
- Knees slightly soft
- Pelvis neutral
- Shoulders back
- Chin tucked
- Ears over shoulders

### Sleeping
| Position | Tips |
|----------|------|
| Back | Pillow under knees |
| Side | Pillow between knees |
| Stomach | Avoid if possible |

## Daily Posture Routine (10 min)

### Morning (5 min)
| Exercise | Time |
|----------|------|
| Cat-cow | 1 min |
| Chin tucks | 1 min |
| Wall angels | 1 min |
| Hip flexor stretch | 1 min each |

### During Work (Every Hour)
- Stand up and move
- Chin tucks (10 reps)
- Shoulder rolls (10 each way)
- Brief walk

### Evening (5 min)
| Exercise | Time |
|----------|------|
| Doorway chest stretch | 1 min |
| Hip flexor stretch | 1 min each |
| Dead bug | 1 min |
| Deep breathing | 1 min |

## Gym Programming for Posture

### Weekly Focus
| Day | Posture-Focused Exercises |
|-----|---------------------------|
| Push | Face pulls, band pull-aparts |
| Pull | Rows with pause, rear delts |
| Legs | Hip flexor stretch, glute work |
| All days | Dead bugs warm-up |

### Rule of Thumb
- 2:1 pull to push ratio for most people
- Face pulls every workout
- Stretch hip flexors daily
- Strengthen glutes 2-3x/week
