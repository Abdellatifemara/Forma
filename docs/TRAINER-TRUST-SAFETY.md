# FitApp Trainer Trust & Safety Policies

## Overview

This document defines the trust and safety framework for the FitApp trainer marketplace. It covers what trainers can/cannot do, how disputes are handled, and how we protect clients from harmful advice.

**This supplements `05-TRAINER-SYSTEM.md` which covers operational details.**

---

# PART 1: TRAINER ADVICE BOUNDARIES

## 1.1 What Trainers CAN Do

### Fitness & Exercise
| Allowed | Examples |
|---------|----------|
| Create workout programs | PPL, upper/lower, full body |
| Modify exercise selection | "Try incline press instead" |
| Adjust sets/reps/rest | "4x8 with 90 sec rest" |
| Provide form guidance | "Keep your back straight" |
| Recommend exercise variations | "Use resistance bands if no weights" |
| Set training frequency | "Train 4 days per week" |
| Suggest active recovery | "Light walk on rest days" |

### Nutrition (General)
| Allowed | Examples |
|---------|----------|
| General nutrition education | "Protein helps muscle recovery" |
| Meal timing suggestions | "Eat carbs around workouts" |
| Macro guidance (if certified) | "Aim for 1.6g protein per kg" |
| Food suggestions | "Try eggs, chicken, fish for protein" |
| Hydration reminders | "Drink 2-3 liters of water daily" |
| Calorie range suggestions | "Try 2000-2200 calories to start" |

### Supplements (After Screening)
| Allowed | Examples |
|---------|----------|
| Recommend GREEN supplements | Protein powder, vitamins |
| Recommend YELLOW supplements | Creatine, omega-3 (with screening) |
| Educational information | "Creatine helps with strength" |
| Timing advice | "Take protein within 2 hours of training" |
| Brand suggestions | "ON Gold Standard is popular in Egypt" |

### Motivation & Accountability
| Allowed | Examples |
|---------|----------|
| Progress check-ins | "How did this week go?" |
| Encouragement | "Great job sticking to the plan!" |
| Goal setting | "Let's aim for 4 workouts this week" |
| Habit building | "Try morning workouts for consistency" |
| Mindset coaching | "Focus on progress, not perfection" |

## 1.2 What Trainers CANNOT Do

### Medical Territory - NEVER ALLOWED

| Forbidden | Why | Alternative |
|-----------|-----|-------------|
| Diagnose conditions | Medical practice | "See a doctor" |
| Prescribe supplements for conditions | Medical advice | "Discuss with your doctor" |
| Recommend ORANGE/RED supplements | Dangerous | Blocked by system |
| Treat injuries | Requires physiotherapist | "See a physio first" |
| Recommend exercises for injury rehab | Medical territory | "Get clearance first" |
| Advise on medical conditions | Practicing medicine | "Work with your doctor on this" |
| Interpret blood tests | Medical expertise | "Discuss results with your doctor" |
| Recommend stopping medications | Dangerous | Never mention medications |

### Nutrition - RESTRICTED

| Forbidden | Why | Alternative |
|-----------|-----|-------------|
| Create medical diet plans | Requires dietitian | General guidance only |
| Treat eating disorders | Mental health | Refer to professional |
| Recommend <1200/1500 cal | Dangerous | System blocks this |
| Diagnose food allergies | Medical | "Get tested if you suspect" |
| Advise on diabetes nutrition | Medical | "Work with your doctor" |
| Create pregnancy nutrition plans | Medical | "Consult your OB/GYN" |

### Professional Boundaries

| Forbidden | Why |
|-----------|-----|
| Personal relationships beyond coaching | Conflict of interest |
| Requesting personal social media | Privacy |
| Communicating outside app | No oversight, payment bypass |
| Requesting payment outside app | Terms violation |
| Selling products directly | Conflict of interest |
| Making guarantees | Unrealistic expectations |

### Dangerous Practices

| Forbidden | Why |
|-----------|-----|
| Recommending steroids/SARMs | Illegal, dangerous |
| Extreme dieting tactics | Health risk |
| "Quick fix" promises | Unrealistic, harmful |
| Overtraining encouragement | Injury risk |
| Ignoring pain complaints | Safety concern |
| Dismissing medical concerns | Liability |

# PART 3: DANGEROUS ADVICE ESCALATION

## 3.1 What Constitutes "Dangerous Advice"

### Severity Level 1: IMMEDIATE ACTION
```
- Recommending steroids, SARMs, or banned substances
- Advising to ignore chest pain/serious symptoms
- Recommending extreme caloric restriction (<800 cal)
- Suggesting exercises that caused injury
- Dismissing signs of eating disorder
- Medical diagnosis attempts
```

### Severity Level 2: INVESTIGATION REQUIRED
```
- Recommending ORANGE/RED supplements without screening
- Pushing excessive training volume (>20 hours/week)
- Ignoring client's reported pain
- Inappropriate modification for injuries
- Recommending exercises contraindicated by screening
- Pressuring unhealthy weight goals
```

### Severity Level 3: WARNING + EDUCATION
```
- Minor form advice errors
- Slightly aggressive calorie recommendations
- Forgetting to check screening results
- General nutrition advice that overreaches
- Recommending supplements without checking history
```

## 3.2 Detection Methods

### Automated Detection (AI-Flagged)

The system monitors trainer messages for:
```
KEYWORDS FLAGGED:
- "steroids", "SARMs", "gear", "juice" (context checked)
- "ignore the pain", "push through"
- Extremely low calorie numbers
- Medical diagnosis language
- Supplement names not in GREEN/YELLOW list
- "Don't worry about the doctor"

PATTERN FLAGGED:
- Multiple clients complaining about same trainer
- Trainer overriding many safety defaults
- High refund request rate
- Client messages mentioning injury after advice
```

## 3.3 Response Protocol

### Severity 1: IMMEDIATE ACTION
```
1. Trainer account suspended immediately
2. All active clients notified
3. Full refunds processed automatically
4. Admin investigation within 24 hours
5. If confirmed:
   - Permanent ban
   - Report to relevant authorities if illegal
   - Public profile removed
```

### Severity 2: INVESTIGATION
```
1. Trainer notified of concern
2. Feature restrictions applied (can't add new clients)
3. Admin reviews within 48 hours
4. If confirmed:
   - First offense: Warning + mandatory education
   - Second offense: 30-day suspension
   - Third offense: Permanent ban
```

### Severity 3: WARNING
```
1. Automated warning sent to trainer
2. Education resources provided
3. Logged for pattern tracking
4. If repeated (3+ times):
   - Escalate to Severity 2
```
